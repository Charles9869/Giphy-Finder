# Giphy-Finder

Basic React application to find gifs using the Giphy API.

## TECHNOLOGIES

- React
- NodeJS and Express
- Giphy API

## RUN LOCALLY

```
$ git clone https://github.com/Charles9869/Giphy-Finder.git

```

```
$ cd server
$ yarn install
$ yarn start
```

```
$ cd client
$ yarn install
$ yarn start
```
